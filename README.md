# MoC textures modding
Creates textures for modding of "Master of Command" video game.

Use your picture and get:
1. Flag
2. Battle flag
3. Flag waving left
4. Flag waving right
5. Camp flag
6. Unit tree tileset

## How to use
1. Copy files.
2. Change image_sourse.png on your flag picture
3. Open moc_flags.ipynb
4. Change your tag in moc_flags.ipynb
5. Run moc_flags.ipynb
6. Get flag assets
7. Open moc_tileset.ipynb
8. Change colors (RGBA)
9. Run moc_tileset.ipynb
10. Get tileset
